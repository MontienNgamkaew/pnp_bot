# บันทึกสรุปการปฏิบัติงานล่าสุด (Work Summary - PNP BOT)

บันทึกสรุปรายการพัฒนาระบบบอตและแก้ไขข้อบกพร่องต่าง ๆ ประจำวันที่ 1 กรกฎาคม 2569

## 🎯 รายละเอียดการอัปเดตและพัฒนาฟีเจอร์

### 1. การติดตั้ง SDK และแก้ไขตรรกะเรื่องปี พ.ศ. / ค.ศ.
* **การติดตั้ง**: ทำการติดตั้งแพ็กเกจ `@google/generative-ai` ลงในโปรเจกต์สำเร็จ
* **แก้ไขบั๊กเรื่องปี (2612 → 2569)**:
  - แก้ไขปัญหา AI แปลงปี พ.ศ. 2569 สับสนจนแสดงผลเป็น พ.ศ. 2612 (ค.ศ. 2069)
  - เพิ่มการระบุวันเวลาปัจจุบันทั้งรูปแบบ พ.ศ. และ ค.ศ. ใน System Prompt ของ AI
  - เพิ่มระบบความปลอดภัย (Safeguard) ในตัวแปลงวันเวลาฝั่งเซิร์ฟเวอร์ โดยจะแปลงปี พ.ศ. ที่เกิน 2400 ให้กลายเป็น ค.ศ. โดยอัตโนมัติก่อนจัดเก็บ

### 2. ฟีเจอร์สรุปเอกสารและรูปภาพด้วย AI (`/สรุป`)
* **สรุปแบบระบุไฟล์เดี่ยว (Reply)**: ผู้ใช้สามารถตอบกลับ (Reply) รูปภาพ หรือ PDF ในกลุ่ม LINE ด้วยคำว่า `/สรุป` เพื่อส่งไฟล์แปลงเป็น Base64 ไปยัง Gemini API สำหรับสรุปเนื้อหาเป็นภาษาไทย
* **สรุปรวมหลายไฟล์ (Multi-file)**: หากพิมพ์คำว่า `/สรุป` โดยไม่มีการกดตอบกลับ บอตจะค้นหาไฟล์ประเภทภาพหรือ PDF ที่อัปโหลดล่าสุดภายในห้องแชทนั้นในระยะเวลา 15 นาทีที่ผ่านมา (สูงสุด 5 ไฟล์) เพื่อดาวน์โหลดและส่งรวมกันให้ AI สรุปเชื่อมโยงความสัมพันธ์เป็นเรื่องเดียวกัน
* **รองรับคำสั่งเฉพาะ**: สนับสนุนรูปแบบ `/สรุป [ข้อกำหนดเพิ่มเติม]` เช่น `/สรุป สรุปเฉพาะตัวเลขค่าใช้จ่าย`

### 3. เพิ่มความทนทานต่อการขัดข้องของ Gemini API (503 Service Unavailable)
* **ระบบ Retry (ลองใหม่)**: เมื่อ Gemini API ตอบกลับเป็นสถานะ 503 หรือ 429 ระบบจะหน่วงเวลา 1.5 วินาทีแล้วทดลองส่งใหม่สูงสุด 2 ครั้งต่อโมเดล
* **ระบบ Fallback (โมเดลสำรอง)**: หากยิงคำสั่งไปที่ `gemini-2.5-flash` แล้วล้มเหลว ระบบจะสลับไปเรียกใช้โมเดลรุ่นเสถียรอย่าง `gemini-1.5-flash` ให้เองโดยอัตโนมัติ

### 4. แก้ไขบั๊กการบันทึกไฟล์ Google Drive (400 Invalid Value)
* **สาเหตุ**: ค้นพบว่า Google Drive API v3 ไม่อนุญาตให้ใช้ฟิลด์ `md5Checksum` มากรองข้อมูลค้นหาในตัวแปร `q` โดยตรง (ทำให้ระบบล่มตอนรับไฟล์)
* **วิธีแก้**: ปรับปรุงให้บอตดึงรายการไฟล์ 100 รายการล่าสุดในโฟลเดอร์ปลายทางพร้อมขอฟิลด์ `md5Checksum` แล้วนำมาเทียบค่าซ้ำทางฝั่งโค้ด JavaScript (Client-side Filter) แทน

### 5. อัปเดตเมนูการช่วยเหลือ (`/help`)
* ปรับปรุงฟังก์ชันแสดงผลข้อความช่วยเหลือ ให้สมาชิกภายใน LINE Group ทราบขั้นตอนการเรียกใช้งานฟีเจอร์ `/สรุป` ทั้งแบบตอบกลับไฟล์เดี่ยวและส่งรวมหลายไฟล์

### 6. ฟีเจอร์รายงานนัดหมายยามเช้าโดย AI (AI Daily Briefing)
* **รวบรวมแจ้งเตือน**: ปรับปรุงระบบแจ้งเตือนยามเช้าให้ทำการรวบรวมนัดหมายทั้งหมดในวันนั้น ๆ ของแต่ละกลุ่ม LINE ส่งรายงานเป็นข้อความเดียวกัน เพื่อป้องกันการสแปมแจ้งเตือนหลายข้อความ
* **เรียบเรียงบทสรุป**: ส่งตารางนัดหมายทั้งหมดไปให้ Gemini API เพื่อเขียนทักทายยามเช้าภาษาไทยที่อบอุ่น อธิบายตารางงาน และให้แง่คิดคำอวยพรเพิ่มพลังบวกเริ่มต้นเช้าวันทำงาน
* **มีระบบสำรองป้องกันระบบล่ม (Fallback)**: ครอบระบบ AI ด้วย Try-Catch หากการเรียก API ขัดข้อง บอตจะหันไปดึงข้อความแจ้งเตือนมาตรฐานแบบเดิมส่งให้สมาชิกกลุ่มโดยอัตโนมัติ ทำให้มั่นใจได้ว่าผู้ใช้จะไม่พลาดการนัดหมายอย่างแน่นอน
* **แก้ไขบั๊กแจ้งเตือนช่วงเที่ยงคืน (Midnight Trigger Bug Fix)**: แก้ไขปัญหาบนระบบที่มักจะจัดเก็บ/แสดงผลชั่วโมงในเวลาเที่ยงคืน (00:00) เป็นชั่วโมงที่ 24 ในฟังก์ชัน `getBangkokDateParts` ส่งผลให้เงื่อนไขเวลาคิดว่าเลย 8 โมงเช้าแล้ว (24 >= 8) จึงทริกเกอร์ส่งแจ้งเตือนยามเช้าตอนเที่ยงคืนและข้ามการเตือนในเวลาเช้าจริง โดยแก้ไขโดยการปรับใช้ modulo 24 `% 24` เพื่อบังคับให้อ่านค่าชั่วโมงที่ถูกต้องเสมอกันทุกสภาพแวดล้อมระบบ

### 7. ระบบโต้ตอบทั่วไปและจัดการนัดหมายด้วยภาษาธรรมชาติ (NLP AI Assistant)
* **การทริกเกอร์บอต**: เพิ่มการดักจับข้อความที่มีคีย์เวิร์ดนำหน้าด้วย `บอต`, `บอท`, `bot` หรือ `@bot` (รองรับการพิมพ์ติดกัน เช่น `บอตเลื่อนนัด` และป้องกันการชนกับศัพท์ภาษาอังกฤษ เช่น `bottle`, `button`)
* **ระบบเข้าใจเจตนา (Intent Classification & NLP)**:
  - ใช้ Gemini API ตรวจสอบเจตนาของผู้ใช้ว่าต้องการ นัดหมายใหม่ (`create`), เลื่อนนัดหมาย (`update`), ยกเลิกนัดหมาย (`cancel`), ขอดูรายการนัดหมาย (`list`), หรือสอบถามทั่วไป (`general`)
  - **การเลื่อนนัด/ยกเลิกนัด (NLP Update & Cancel)**: บอตจะดึงนัดหมายที่เปิดใช้งานอยู่ทั้งหมดมาวิเคราะห์ร่วมกับคำขอ เพื่อจับคู่นัดหมายที่ต้องการเลื่อน/ยกเลิกโดยอัตโนมัติ (ผ่านหัวข้อ, วันเวลา หรือรหัส) นอกจากนี้ยังสนับสนุนคำสั่งเฉพาะ `/เลื่อนนัด` และ `/reschedule` ตามด้วยข้อความภาษาธรรมชาติเพื่อบังคับวิเคราะห์แบบเลื่อนเวลานัดหมายโดยตรงทันที
  - **การคุยโต้ตอบทั่วไป (AI Group Chat Assistant)**: หากไม่ใช่งานเกี่ยวกับนัดหมาย บอตจะตอบคำถามหรือคุยโต้ตอบเป็นภาษาไทยที่เป็นมิตรโดยใช้ Gemini API
  - **ระบบกู้คืนแจ้งเตือน (Reschedule Reset)**: หากมีการเลื่อนเวลานัดหมาย บอตจะทำการรีเซ็ตสถานะการส่งแจ้งเตือนยามเช้าและเตือน 10 นาทีล่าสุดให้กลับเป็นยังไม่ได้ส่งโดยอัตโนมัติ เพื่อให้ระบบทำการแจ้งเตือนอีกครั้งในวันเวลาใหม่ได้อย่างถูกต้อง
